## わかったこと
`git log --oneline`でメッセージのみのgit logを閲覧できる
`git log --oneline --graph`グラフ形式で見ることができる
`git show {コミットid}`過去のコミットとの差分を表示できる
`git restore --staged {ファイル名}`　指定したファイルをステージ前の状態に戻す
`git restore --source=HEAD^ .`ようにするに直前のコミットを指定している
```
#使用例
git restore --source=HEAD~2 file.txt   # 2つ前のコミット
git restore --source=commitID file.txt # コミットID指定
git restore --source=main file.txt     # mainブランチ上の最新
```
`git branch -D {ブランチ名}`ブランチの削除
`git stash`で変更分を一時的に保存することができる。
`git stach list`でstashした変更を一覧で確認できる。
`git stash apply 0 `で直近のstashの変更を適用する。stashは残る
`git stash pop`直近の変更を適用してstash を削除
`git remote -v`現在のリモートリポジトリの情報を確認できる
