## わかったこと
gridレイアウトを作る時はまず全体を包む
グリッドレイアウトではこの要素をグリッドアイテムという。

`margin: 0 auto;` 中央寄せの定番

```
.itemA {
  grid-column-start: 1;
  grid-column-end: 2;
  grid-row-start: 1;
  grid-row-end: 3;
}

#短縮形
.itemA {
  grid-column: 1 / 2;
  grid-row: 1 / 3;
}
```
`grid-column: 1 / 2'の様に隣あっているものを表す場合は`grid-column: 1`と記述することができる。
```
.itemA {
  grid-column: 1 / 2;
  grid-row: 1 / 3;
}
```
`reset.css`ファイルは`style.css`ファイルよりも**前に置くことが大事！**
